## Descripción

Portafolio personal desarrollado con HTML5 y CSS3 puro. Presenta mis proyectos, habilidades y formas de contacto, con un diseño responsivo adaptado a desktop, tablet y móvil.

---

## Tecnologías utilizadas

- HTML5
- CSS3 (Flexbox, Media Queries)
- Git & GitHub
- GitHub Pages

---

## Características

- Diseño responsivo para 1440px, 768px y 320px
- Sección de proyectos destacados con links a demo en vivo y repositorio
- Sección de habilidades y herramientas
- Footer con links a redes sociales y contacto

---

## Proyectos incluidos

### Página web de una cafetería

Landing page responsiva para una cafetería siguiendo un brief de diseño. Incluye estructura semántica, maquetación con Flexbox y formulario de contacto.
